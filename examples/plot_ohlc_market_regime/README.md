# plot_ohlc_market_regime

Representative OHLC plotting example for market-regime inspection.

## What this demonstrates

- native `ohlc` trace support in `vsl.plot`
- day-by-day open/high/low/close bars
- a practical chart used in financial and trading dashboards

## Run

```sh
v run examples/plot_ohlc_market_regime/main.v
```
