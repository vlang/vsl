# OpenCL Kernel Parameters Example

This example demonstrates configuring and passing parameters to OpenCL kernels
through the VCL module.

## Run

```sh
v run examples/vcl_opencl_kernel_params/main.v
```

## Notes

- Requires OpenCL runtime and compatible device.
- If your environment uses dynamic OpenCL loading flags,
  run with the same flags used in your local VSL setup.
